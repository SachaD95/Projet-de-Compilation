# Projet Compilateur : Fonctions, Tableaux et Structures

## Présentation du projet
L'objectif de ce projet est de réaliser un compilateur prenant en charge l'utilisation de fonctions, de tableaux et de structures.

## Organisation du dépôt
Le développement a été réparti au sein de l'équipe : il existe une branche spécifique pour chacune des fonctionnalités, implémentée individuellement par l'un d'entre nous. 

Nous avons ensuite tenté d'intégrer toutes ces fonctionnalités ensemble. Les résultats de ces intégrations se trouvent sur les branches suivantes :

* **`Feat/Tab_struct_fonct`** : Tentative d'intégration de toutes les fonctionnalités (Tableaux + Structures + Fonctions). Sur cette branche, **seul le Pretty Printer (PP) est opérationnel**.
* **`Feat/tab_fonct`** : Intégration ciblée sur les tableaux et les fonctions. Sur cette branche, **le Pretty Printer ET la génération de code assembleur sont opérationnels**.
